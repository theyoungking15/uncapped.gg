# Features and Changes

## Features (baseline)
- PC Builder section scaffold with summary panel, tabbed categories, action buttons, and mobile drawer/sticky bar UI.
- Collection-based product loading per category via `?view=ucp-pc-builder-json` endpoints.
- Bundle rules support with embedded fallback JSON and fetch-all pagination via `bundle_rules_endpoint`.
- Config JSON bootstrapping (`#ucp-pcb-config`) for endpoints, Messenger URL, quote contact endpoint, and perf preview toggle.
- Brand logo resolution using metaobject -> shop brand -> theme logo priority.
- Performance Preview card UI with preset pills and FPS display (toggle via section setting).
- Filter controls per category (brand, socket, chipset, GPU brand, board partner, form factor, watts, cooler type, case form factor) plus price slider.
- Selection logic with variant handling, multi-pick tabs (case fans, other), and dependency enforcement.
- Summary rendering with subtotal and bundle savings display.
- Add-to-cart flow that posts selected variants (including multi-qty items).
- Share build link: generate URL with selected variant IDs; parse and auto-apply selections from URL.
- Request Quote flow: quote code generation, modal UI, build link copy, and Messenger handoff.
- PDF generation for the build summary with pricing and bundle savings.
- Google Apps Script build logging for key events (add_to_cart, generate_pdf, share, request_quote).
- Mobile UX helpers: sticky tabs on scroll, summary drawer overlay, and mobile action buttons wired to desktop actions.
